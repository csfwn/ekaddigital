# Ekad Digital

Landing page Bahasa Melayu untuk kad kahwin digital, dibina dengan **TanStack React + Tailwind CSS** dan bergaya **Tailwind UI**.

## Stack
- React 18
- TypeScript
- Vite
- TanStack React Query
- Tailwind CSS
- Lucide icons

## Cara run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
